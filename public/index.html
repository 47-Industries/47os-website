Install in 60 seconds.

One command. Fresh Linux Mint. That's all you need.

git clone https://github.com/phantom47m/47os-rice.git && cd 47os-rice && ./install.sh
  1. Open a terminal on your Linux Mint machine
  2. Paste the command above and hit Enter
  3. Type y to confirm, enter your password
  4. Select No on the macchanger prompt
  5. Let it finish, reboot, done
